# ☕ Coffee Shop Sales Analysis Dashboard

An interactive Excel-based dashboard designed to analyze coffee shop sales data, uncover trends, and provide actionable insights for better business decisions.

---

## 📌 Project Overview

This project demonstrates how Microsoft Excel can be used as a powerful data analysis and visualization tool.

It focuses on:
- Identifying sales trends over time  
- Understanding customer purchasing behavior  
- Highlighting top-performing products & locations  
- Supporting data-driven decision-making  

---

## 📊 Dashboard Features

- **Sales Trend Analysis** – Track daily, weekly, and monthly sales performance  
- **Product Performance** – Identify best-selling and underperforming items  
- **Location Insights** – Compare sales across multiple store locations  
- **Peak Hours & Days** – Discover when customers buy the most  
- **Interactive Filters** – Slice data by date, product category, or location  

---

## 🛠 Tools & Technologies

- Microsoft Excel (Pivot Tables, Charts, Slicers, Conditional Formatting)  
- Power Query (Data Cleaning & Transformation)  
- Data Source – Sample transactional dataset (CSV/Excel)
